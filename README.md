# telecom-analysis

El objetivo del proyecto es identificar patrones de uso de los clientes y detectar valores atípicos para comprender que segmentos son los que muestran necesidades diferenciadas. 
Con el fin de mejorar la oferta comercial y mejorar la experiencia del usuario. 

Los datasets utilizados son: 
- plans.csv
- users.csv
- usage.csv

Lo primero que se hizo dentro del análisis fue limpiar e integrar las bases de datos provenientes de las tres fuentes distintas. 
Después se aplicó técnicas de validación para estandarizar los tipos de datos y la detección de valores inconsistentes. 
También se detectaron outliers y valores atípicos mediante métodos estadísticos y visuales como histogramas y diagramas de caja. 
Se crearon segmentos por edad y por los comportamientos en el uso de servicios móviles de los clientes, con el fin de construir perfiles demográficos del uso de llamadas para ver las tendencias y ver cómo se pueden mejorar los servicios. 

Se visualizaron los diferentes segmentos para extraer insights comerciales relevantes de los patrones de los clientes en el uso de los servicios móviles.
